Current Requirements:
1. Any number of players can play the game
2. The board can numbers from 1 to N(configurable by clinet).
3. Dice can give number anything from 1 to anything.
4. Multiple dices can be there in one game and size of each dice can vary.
5. Game should start only when a person gets 1 or 6 
6. Allow a random way to create snakes and ladders and also allow the manual way of creating snakes and ladders(user can choose between the two)
7. Each player can have n number of pieces(a player is winner only when he is able to place all the n pieces at the end of the board)
